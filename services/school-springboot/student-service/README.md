# Student Service

This service manages student-related operations in the Altera School Management System.

## Features
- Student registration and profile management
- Student enrollment tracking
- Academic history
- Parent-student relationships
- Student performance analytics

## Technologies
- Spring Boot
- Spring Data JPA
- MySQL/PostgreSQL

## API Endpoints
- GET /students
- POST /students
- PUT /students/{id}
- DELETE /students/{id}
- GET /students/{id}/courses
- GET /students/{id}/grades
