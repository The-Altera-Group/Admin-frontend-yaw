# Authentication Service

This service handles user authentication and authorization for the Altera School Management System.

## Features
- User registration and login
- JWT token management
- Role-based access control
- Password reset functionality

## Technologies
- Spring Boot
- Spring Security
- JWT
- MySQL/PostgreSQL

## API Endpoints
- POST /auth/login
- POST /auth/register
- POST /auth/refresh
- POST /auth/logout
- POST /auth/forgot-password
