# Teacher Service

This service manages teacher-related operations in the Altera School Management System.

## Features
- Teacher profile management
- Course assignments
- Class management
- Teacher scheduling
- Performance tracking

## Technologies
- Spring Boot
- Spring Data JPA
- MySQL/PostgreSQL

## API Endpoints
- GET /teachers
- POST /teachers
- PUT /teachers/{id}
- DELETE /teachers/{id}
- GET /teachers/{id}/courses
- GET /teachers/{id}/schedule
