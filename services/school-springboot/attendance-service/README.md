# Attendance Service

This service manages attendance tracking in the Altera School Management System.

## Features
- Daily attendance recording
- Attendance reports
- Absence notifications
- Attendance analytics
- Parent notifications

## Technologies
- Spring Boot
- Spring Data JPA
- MySQL/PostgreSQL

## API Endpoints
- POST /attendance/record
- GET /attendance/student/{id}
- GET /attendance/class/{id}
- GET /attendance/reports
- PUT /attendance/{id}
