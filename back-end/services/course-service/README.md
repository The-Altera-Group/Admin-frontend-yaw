# Course Service

This service manages course-related operations in the Altera School Management System.

## Features
- Course creation and management
- Subject management
- Class scheduling
- Course enrollment
- Curriculum management

## Technologies
- Spring Boot
- Spring Data JPA
- MySQL/PostgreSQL

## API Endpoints
- GET /courses
- POST /courses
- PUT /courses/{id}
- DELETE /courses/{id}
- GET /courses/{id}/students
- POST /courses/{id}/enroll
